BAGAIMANA CARA KAMU MEMULAI MENJADI PROGRAMMER PEMULA


1. step pertama

yang harus kamu pelajari adalah mengenali html dan css terlebih dahulu. kenapa harus belajar html dan css dulu karena hal dasar untuk menjadi web development itu menguasai html dan css dahulu karena itu paling dasar nya.
owhhh iya saya hanya ingin sharing sedikit, menurut saya pribadi menjadi seorang
programmer harus bertahap, jadi ada 4 developer diantaranya ada web development, ada android developer,
game development, dan software developer. nah yang harus kalian kuasai paling dasar itu menurut saya adalah
menguasai web developer terlebih dahulu selebih nya tergantung pilihan kalian. jadi ketika kalian sudah menguasai
satu bidang ketika ingin pindah ke bidang yang lain tidak akan telalu susah itu menurut saya ya.

2. ke dua

kalo kamu sudah menguasai html dan css selanjutnya kamu bisa ngelanjutin ke bahasa pemograman javascript atau php kalau
kata saya sih mending langsung ke javascript aja karena kalo udah bisa menguasai javascript pindah ke php tidak terlalu susah

3. ketiga

abis itu belajar apalagi bang?, nah kalian lanjut belajar github dan unix command line. github emang buat apaan bang??, terus
